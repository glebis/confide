# Contributors

CONFIDE is a **community / citizen-science** project — built and maintained by
**volunteers**, not a funded lab. Its trustworthiness comes from people inspecting,
correcting, and extending it in the open. This file credits that work. If you contributed
and are not listed (or want to be listed differently, or not at all), open a PR or let us
know — see `CONTRIBUTING.md`.

> Why this file exists: open recognition of contributors is a core citizen-science norm
> (ECSA Ten Principles of Citizen Science, Principle 8 — "citizen scientists are
> acknowledged in project results"). We try to live up to it.

## How contributions are credited

- The authoritative record of code contributions is the **git history**
  (`git shortlog -sne`); this file is the human-readable summary.
- Substantive contributors to a write-up or release will be offered **co-authorship /
  named credit** on that artifact, not just a commit line.
- Data, gold corrections, and annotation work are first-class contributions, credited the
  same as code.

_Licensing: contributions are accepted under the repository's licenses — code MIT, data &
docs CC-BY-4.0 (see `LICENSE`). By contributing you agree your work is released under
those terms; see `CLA.md`._
